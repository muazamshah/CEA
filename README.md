# CEA Website

Small static website for the CEA project.

## Overview
A minimal single-page site delivered with `index.html`, `style.css`, and `script.js`.

## Features
- Simple responsive layout
- Basic JavaScript interactivity

## Getting started
Open the site directly in your browser by double-clicking `index.html` or run a local server.

Quick local server (Python 3):

```bash
python -m http.server 8000
# then open http://localhost:8000
```

Or with Node (if you have `serve` installed):

```bash
npx serve .
```

## Project structure
- `index.html` — Main page
- `style.css` — Styles
- `script.js` — Client-side JavaScript
